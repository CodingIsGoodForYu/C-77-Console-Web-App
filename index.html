Visual Studio

Visual Studio, also known as Microsoft Visual Studio and VS, is an integrated development environment for Microsoft Windows. It is a tool for writing computer programs, websites, web apps, and web services. It includes a code editor, debugger, GUI design tool, and database schema designer, and supports most major revision control systems.

Console Screen

Simply put, console means input from the keyboard and output to the screen. On another level it can mean running a program in a specific window called a “console”. Visual Studio offers this functionality.
